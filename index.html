Project Idea

Overview of project

My project is a mass sorting system for files. The project will display a file visually and allows the user to drag that file into sub folders. The program must allow the users to select multiple files, create multiple folders, delete folders, view images and videos via a thumbnail and allow users to drag their files into folders. The program should be easy to use for someone who has basic knowledge of computer systems. The program must also be quick. The user must have the ability to select the directory of the files waiting to be sorted. The motivation behind this program is the vast amount of photos, videos and other files being stored on pc’s around the world and a lack of sorting options limits the user forcing them to make folders for each one in the windows explorer. This program is aimed at people who store movies and photos on their pc’s and have them scattered everywhere. The program fills in the much-needed sorting system that windows explorer lacks.

Detailed Description

The program starts of by asking the user to select a directory for the files they are wanting to sort. This will open a browse popup and lets the user select the folder. Once the folder is selected the program displays each file as a thumbnail in the middle of the screen, below the file is the name of it. Above this there is a scrollbar for each of the folders the user can drag the file into, these folders are loaded from a source folder defined by the user. The user will also have the ability to create a new folder, once created it will be added to the top bar. This bar will expand as the user adds a folder. The program will also have the ability to create default folders for the user, for example if the user is sorting movies it will make folders for each genre romance, sci-fi, comedy, animation… The program will continue to display a file until all files are moved out of the folder. The program will also allow the user to automatically sort by file types for example if there are multiple text files it will move them into a folder label “Text Files”. The program will follow the basic layout functions set by windows such as the layout of minimize, maximize and close buttons and the icons from windows 10 such as folders and thumbnails. The program will not be resource heavy and should be able to run on any laptop or pc despite it’s processing power. Users will have the ability to load previous folder layouts and save them.

Tools and Technology Required

This program will require a windows system and java on the users computer. To develop the soloution eclipse will need to be installed and use of github to store the code and retrieve it. The program will not require a powerfull computer to run and should be able to run on most modern computers.

Outcome

When the program is finished it should be able to successfully let the user fix their problems of messy folders with no organization. If the program functions as intended being able to find a movie, image, document or any other file type will be made a lot easier than previously and the user will be able to move the categories around and store them in different locations and use them with other computer systems like home pc’s or ipods.
